# PhoneGap

  - Be sure to have [node.js](https://nodejs.org/en/) and [git](https://git-scm.com/downloads) installed.
  - Install the PhoneGap CLI by using the command $ npm install -g phonegap@latest.
  - Download the Mobile App.
  - Create an app by using the command $ phonegap create MyApp.
  - Open the folder created and replace the www folder with the one in the repository.
  - Open the command line and go to the App folder and use the command $ phonegap serve.
  - Open the moibile app and insert the ip address.
